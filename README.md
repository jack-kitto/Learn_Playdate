# Learn Playdate

## 1. Editor Setup

### 1.1 Requirements:
- Bear:
- Make

Install with:
```sh
brew install bear make
```
### 1.2 Generate Compile Commands:

Use this command to generate `compile_commands.json` file for clangd
```sh
bear -- make
```

## 2. Build
```sh
./start.sh
```
